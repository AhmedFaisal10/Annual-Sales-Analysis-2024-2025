# Annual Sales Analysis 2024–2025

This project provides a full annual sales analysis for the company for the years **2024–2025**, combining both:

* **Raw Excel sales data**
* **Python-based Exploratory Data Analysis (EDA)** with visualization

The goal is to deliver clear insights into product performance, customer contribution, and yearly sales trends.

## How to Run

1. Clone the repository:


2. Install dependencies:

```
pip install pandas numpy matplotlib
```

3. Open the notebook:

```
jupyter notebook analysis/sales_analysis.ipynb
```
